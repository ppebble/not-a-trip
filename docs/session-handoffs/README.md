# Session Handoffs

이 디렉터리는 긴 작업 세션을 끊고 새 세션에서 재개할 때 필요한 최소 문서를 모아둡니다.

## 파일 구성

- `todo.md`
  - 다음 세션에서 바로 이어갈 작업 목록
- `workflow.md`
  - requirements / design / tasks 기반 작업 프로세스
- `status.md`
  - 현재까지 완료 상태, 최근 Git 흐름, 재개 시 확인할 포인트

## 사용 순서

1. `status.md`로 현재 상태 확인
2. `workflow.md`로 작업 방식 확인
3. `todo.md`에서 다음 실행 항목 선택
