# Not A Trip - Spec 로드맵

## 서비스 방향성

**As-Is**: 성지순례 정보를 보고, 편의시설을 찾고, 게시판에서 소통하는 여행 정보 사이트

**To-Be**: 오타쿠들의 **'덕질 퀘스트(Quest)'**이자, 본인이 다녀온 성지를 수집하고 전시하는 **'트로피 룸(Trophy Room) 및 아카이브'**

## 🎯 런칭 전략: 선택과 집중

### 타겟 작품 (초기 3개)

1. **너의 이름은** - 글로벌 인지도 최고
2. **슬램덩크** - 가마쿠라/에노시마 집중
3. **봇치 더 록** - 최신작, 시모키타자와 집중

### 타겟 지역 (초기 2개)

1. **도쿄** (신주쿠, 시부야, 시모키타자와)
2. **가마쿠라/에노시마**

## ⚠️ 리스크 관리

### 저작권 문제 우회

- 애니메이션 원본 캡쳐 대신 **유저 인증샷**을 메인 콘텐츠로 활용
- 스케치/선화로 가공하거나, 유저가 직접 찍은 비교샷 활용

### SEO 전략

- Next.js SSR/SSG 적극 활용
- "너의 이름은 계단 위치", "봇치더록 라이브하우스" 등 검색 키워드 최적화
