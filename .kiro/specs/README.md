# Not A Trip - Spec 우선순위 로드맵

## 서비스 방향성

**As-Is**: 성지순례 정보를 보고, 편의시설을 찾고, 게시판에서 소통하는 여행 정보 사이트

**To-Be**: 오타쿠들의 **'덕질 퀘스트(Quest)'**이자, 본인이 다녀온 성지를 수집하고 전시하는 **'트로피 룸(Trophy Room) 및 아카이브'**

---

## 🎯 Phase 1: MVP 핵심 기능 (최우선)

### 1. [pilgrimage-checkin](./pilgrimage-checkin/) - 성지순례 인증 시스템

- 씬 인증샷 업로드 및 비교 뷰어
- 유저 프로필 및 트로피 룸
- 뱃지/업적 시스템
- **기존 자유게시판 → 인증 갤러리로 전환**

### 2. [mobile-first-ux](./mobile-first-ux/) - 모바일 퍼스트 UX

- 모바일 지도 UX 개선 (Bottom Sheet, 제스처)
- 빠른 인증 플로우
- PWA 지원
- 성능 최적화

---

## 🚀 Phase 2: 성장 기능

### 3. [spot-report-wiki](./spot-report-wiki/) - 성지 제보 시스템

- 유저 성지 제보 기능
- 최초 제보자 명예 시스템
- 스팟 상태 관리 (소실/변경 신고)
- 관리자 검토 시스템

### 4. [pilgrimage-route](./pilgrimage-route/) - 성지순례 코스/루트

- 코스 생성 및 공유
- 코스 따라가기 모드
- 추천 코스 큐레이션

---

## 🎁 Phase 3: 부가 기능

### 5. [otaku-facilities](./otaku-facilities/) - 덕후 친화적 편의시설

- 코인 로커, 혼밥 식당, 굿즈샵 정보
- 편의시설 유저 제보

## 🎯 런칭 전략: 선택과 집중

### 타겟 작품 (초기 3개)

1. **너의 이름은** - 글로벌 인지도 최고
2. **슬램덩크** - 가마쿠라/에노시마 집중
3. **봇치 더 록** - 최신작, 시모키타자와 집중

### 타겟 지역 (초기 2개)

1. **도쿄** (신주쿠, 시부야, 시모키타자와)
2. **가마쿠라/에노시마**

## ⚠️ 리스크 관리

### 저작권 문제 우회

- 애니메이션 원본 캡쳐 대신 **유저 인증샷**을 메인 콘텐츠로 활용
- 스케치/선화로 가공하거나, 유저가 직접 찍은 비교샷 활용

### SEO 전략

- Next.js SSR/SSG 적극 활용
- "너의 이름은 계단 위치", "봇치더록 라이브하우스" 등 검색 키워드 최적화
