# Steering 시스템 메타 설정

이 디렉토리의 모든 steering 파일들은 Kiro 실행 시 자동으로 포함됩니다.

## 포함된 가이드라인

1. **commit-guidelines.md** - 커밋 메시지 작성 필수 규칙
2. **task-execution-rules.md** - Task 실행 필수 프로세스
3. **git-workflow.md** - Git 워크플로우 전체 가이드

## 적용 범위

- 모든 커밋 작성 시
- 모든 Task 실행 시
- 모든 브랜치 생성 시
- 모든 PR 생성 시

이 가이드라인들을 준수하지 않으면 작업이 거부될 수 있습니다.
